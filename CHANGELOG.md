- ✨ 12宫紫微斗数盘展示功能

- ✨ 完整的参数配置页面（出生时间、性别、晚子时、飞星自化、自定义四化、闰月分界）
- ✨ 参数页面（星曜参数表格、宫位-星曜-地支表格）
- ✨ 宫位点击高亮功能
- ✨ 四化星高亮显示
- ✨ 自化/对宫化入图标显示
- ✨ 格局信息展示
- ✨ 国际化支持（中文/英文）
- ✨ 响应式设计，支持移动端和桌面端
- ✨ API 代理服务器示例代码
